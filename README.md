# DurakOffline

Классическая карточная игра "Дурак" для двух игроков на одном устройстве. Игроки физически передают устройство друг другу, чтобы не видеть карты противника.

## Особенности
- Полная колода 36 карт (6,7,8,9,10,В,Д,К,Т)
- Классические правила (атака, защита, подкидывание, бита)
- Режим Hot Seat (игроки передают устроство друг другу)
- Удобный и уродливый интерфейс (интуитивно понятные кнопки)
- Подсветка доступных ходов (зеленым цветом)
- Автоматические подсказки (какие карты можно/нельзя подкинуть)

## Правила игры
- Начало игры: каждому игроку раздаётся по 6 карт
- Козырь: определяется открытой картой в колоде
- Атака: игрок выкладывает карту на стол
- Защита: второй игрок должен побить карту:
   - Той же мастью, но старше
   - Козырем (если атакующая карта не козырная)
- Подкидывание: атакующий может подкинуть карты тех же достоинств, которые уже есть на столе
- Бита: если все карты отбиты, атакующий завершает ход
- Взять карты: если защитник не может отбиться, он забирает все карты со стола
- Победа: выигрывает тот, у кого не осталось карт
